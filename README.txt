Car Factory Manager

Car Factory Manager este un joc distractiv ce pune la incercare cunostintele de cultura generala ale utilizatorului. Aplicatia prezinta o interfata atractiva si colorata, intuitiva. Scopul jocului este de a construi si vinde masini, dezvoltand astfel o afacere profitabila. Dispune de doua sisteme de imbunatatire si marire a profitului: un sistem de imbunatatire a calitatii ce creste calitatea cu o unitate la fiecare upgrade, astfel marind costul construirii unei masini si profitul, si un sistem de crestere a profitului, ce creste profitul cu 20 % din valoarea initiala la fiecare upgrade. Pentru a realiza upgrade-ul la calitate, utilizatorul trebuie sa rezolve un quiz de 5 intrebari. Acesta dispune de 12 categorii, fiecare cu mai multe seturi de intrebari. Rezolvarea acestora presupune raspunderea la minim 4 dintre acestea, corect. Fiecate set de intrebari poate fi accesat de 2 ori.


Tehnologii folosite:
